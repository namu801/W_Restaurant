# 식당 데이터 작성 가이드

`restaurant-template.csv`를 구글 시트나 엑셀로 열어서 채워주세요 (구글 시트 기준: 파일 → 가져오기 → 업로드).
2번째 줄(**예시** 행)은 실제로 작동하는 예시니 참고만 하고, 3번째 줄부터(1~20번)를 채워주시면 됩니다.

## 꼭 채워야 하는 항목 (★)

| 컬럼 | 설명 |
|---|---|
| ★ 식당명 | |
| ★ 카테고리 | 짧은 소개 (예: 한정식, 이자카야, 이탈리안) |
| ★ 음식 종류 | 아래 8개 중 해당하는 것. 여러 개면 **" / "**로 구분 (쉼표 X)<br>`한식 / 양식 / 일식 / 중식 / 고기·구이 / 해산물 / 와인·주류 메뉴 / 브런치·카페` |
| ★ 지역 | `강남역` / `신논현역` / `논현역` 중 하나만 |
| ★ 주소 | |
| ★ 1인당 최소·최대 예산 | 숫자만 (원 단위, 콤마 없이. 예: `45000`) |
| ★ 최소·최대 인원 | 이 식당이 감당 가능한 인원 범위 |
| ★ 룸 있음? | `Y` 또는 `N` |
| ★ 예약 방식 | `예약 없이 가능` / `전화 예약 필요` / `예약이 어려운 편` / `예약 안 받음` |

| ★ 주의할 점 | 한두 문장. 없으면 `특별히 없음` |

## 권장 항목 (비워두면 제가 중간값으로 채워요)

이 항목들은 실제로 가보신 느낌이 들어가는 부분이라 채워주시면 훨씬 정확해지지만, 모르면 비워두셔도 괜찮습니다.

| 컬럼 | 선택지 |
|---|---|
| 어울리는 관계 | `친한 친구` `친구 모임` `직장 동료` `학교 선후배` `직장 선배·상사` `가족·친척` `기타` — 여러 개 가능, `" / "`로 구분 |
| 분위기 | `편안한 캐주얼` `무난한 균형` `적당히 격식 있음(대접)` `격식 있고 차분함` — 여러 개 가능 |
| 대화하기 좋은 정도 | `상` `중` `하` — 상 = 조용해서 대화하기 편함 |
| 대접하는 느낌 | `상` `중` `하` — 상 = 격식 있게 대접하는 느낌이 강함 |
| 서비스 안정감 | `상` `중` `하` — 상 = 응대가 안정적이고 편차가 적음 |
| 역에서 가까운 정도 | `상` `중` `하` — 상 = 역에서 가까움 |
| 소음 수준 | `조용함` `보통` `활기있음` |
| 좌석 형태 | `오픈형` `넓은 간격` `반분리형` `독립 룸` |
| 공간 여유로움 | `상` `중` `하` — 상 = 좌석·테이블 간격이 넉넉함 |

## 룸이 있을 때만 (룸 있음? = Y일 때)

- 룸 최소 인원 / 룸 최소 주문 금액(원) / 룸 이용시간 제한(분) — 모르면 비워두세요.

## 주차 가능할 때만 (주차 가능? = Y일 때)

- 주차 방식: `셀프` / `발렛` / `제휴 주차장`

## 그 외

- **정보 확인 날짜**: 직접 가보거나 전화로 확인한 날짜 (`2026-07-16` 형식). 비워두면 오늘 날짜로 넣을게요.
- **네이버지도 / 카카오맵 링크**: 실제 업체 링크를 알면 붙여넣어 주세요. 모르면 비워두셔도 되고, 그러면 이름+주소로 검색 링크를 자동으로 만들어드려요.
- **사진 파일 개수**: 이 식당 사진을 몇 장 준비했는지 숫자만 적어주세요 (아래 사진 규칙 참고).

---

## 사진 전달 규칙

1. 식당 하나당 **1~5장** 정도면 충분해요. 가로(landscape) 사진 위주로 준비해주시면 카드 레이아웃에 잘 맞아요.
2. 파일명은 **`식당명_번호.jpg`** 형식으로 맞춰주세요. 시트의 "식당명" 컬럼과 **완전히 똑같이** 적어주셔야 제가 매칭할 수 있어요.
   - 예: `온기한상 강남_1.jpg`, `온기한상 강남_2.jpg`, `온기한상 강남_3.jpg`
   - 번호가 **1번인 사진이 대표 사진(썸네일)** 으로 쓰여요.
3. 특수문자(`&`, `#`, 이모지 등)는 파일명에 넣지 말아주세요. 한글·영문·숫자·공백·언더바(`_`)만 사용해주세요.
4. 파일 하나당 **5MB 이하** 권장 (용량이 크면 웹에서 느려져요). 스마트폰으로 찍은 원본이면 대부분 괜찮은 크기예요.
5. 전달 방법은 편하신 대로:
   - 구글 드라이브 폴더 만들어서 공유 링크 주시기, 또는
   - 사진 파일을 압축(zip)해서 전달, 또는
   - 프로젝트에 저장하는 법을 아시면 `public/places/` 폴더에 직접 넣어주셔도 돼요 (로고 넣으셨던 것과 같은 방식이에요).

사진과 시트를 함께 주시면, 제가 파일명과 시트의 식당명을 매칭해서 코드에 반영할게요.
