# 용산·서울역권 식당 19곳 — 재검증 결과 및 출처 (2026-07-20)

`꿀꺽`(폐업)과 `유한`(한남동 소재로 접근성 기준에서 벗어나 제외)을 삭제하고, `고메드용산`·`TAA` 2곳을
추가했습니다. 나머지는 캐치테이블·다이닝코드·테이블링·블로그를 다시 교차 검증하고, 새로 추가된
"영업시간" 항목도 채웠습니다.

## 이번 재검증에서 실제로 바뀐 것

- **호우섬 용산아이파크몰점**: 예약 방식을 "전화 예약 필요" → **"예약 안 받음"**으로 정정. 온라인 예약 채널이 확인되지 않고, 여러 후기가 "현장 웨이팅 등록만 가능"이라고 명시함.
- **쇼니노**: 예약 방식을 "예약이 어려운 편" → **"전화 예약 필요"**로 정정. 캐치테이블 예약이 실제로 열려 있고 "예약도 편하고 친절했다"는 후기 확인. (단, 일부 블로그는 "새로고침 끝에 잡았다"는 반대 신호도 있어 완전히 일치하진 않음.)
- **양복점 용산점**: 예약 방식을 "예약이 어려운 편" → **"전화 예약 필요"**로 정정. 캐치테이블·테이블링 온라인 예약 채널이 실제 운영 중임을 확인. 또한 **룸 있음(2층 입식 룸)**을 새로 확인해 추가.
- **아그라 용산아이파크몰점**: **룸 있음 Y → N**으로 정정 (개별 프라이빗룸 없이 단체석만 가능).
- **일일향 용산점**: 발렛비를 3천원/3만원 혼선 → **3,000원**으로 확정.
- **버누드**: 예산을 4만~7만원(추정) → **1.8만~3.5만원**으로 정정 (원문 직접 확인, "KRW 18,000~35,000").
- **포이키친**: 예산을 사용자 확인 기준 **1만~4만원**으로 확정.
- **버누드**: 영업시간을 **매일 17:00-24:00**으로, 주차는 "매장 자체 공간 없음 · 인근 민영 주차장 이용"으로 확정.
- **우나 용산본점**: 영업시간을 사용자 확인 기준 **매일 11:30-22:00(브레이크타임 15:00-17:00)**으로 확정.
